# 3D Cookie Co

Generate custom 3D-printable cookie cutters from AI-generated designs.

## Features
- AI-powered design generation using DALL-E
- PNG to STL conversion using SimpleCookie
- Download cutter and stamp STL files

## Built with
- React + Vite
- Express.js
- OpenAI DALL-E
- SimpleCookie (Potrace + OpenSCAD)
